# Compiladores

Los lenguajes de programacion al igual que el lenguaje humano son notaciones que facilitan nuestra comunicación, pero a diferencia del lenguaje humano este ayuda a establecer una relación con las máquinas (computadoras). 

Ambos tienen 3 características principales:

- Alfabeto: El conjunto de caracteres utilizados en el lenguaje, como el alfabeto en español, inglés, el kanji, hiragana y katakana del japonés.
- Vocabulario: Las palabras válidas que lo conforman
- Sintaxis: La validez de la palabras que se forman
- Semantica: La coherencia que tienen las palabras formadas
